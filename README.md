# react-music-player
music player by react

## Install
```
git clone https://github.com/soolx/react-music-player.git
cd react-music-player
```
> Use NPM

```
npm install
```

> Use NPM

```
yarn
```

## Start
> Use NPM

```
npm start
```

> Use NPM

```
yarn start
```

## Build
> Use NPM

```
npm run build
```

> Use NPM

```
yarn build
```

## Deploy
> Use NPM

```
npm run deploy
```

> Use NPM

```
yarn deploy
```
